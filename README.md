# Simple Board

Simple Board是一款使用Flutter开发的画板程序，拥有矩形、圆形、椭圆型、三角形等多种形状，并提供对绘制图形的复制、粘贴和对操作的撤销功能。

### 2023.5.15

实现了一个*实验性*的画板，微调了界面的配色，加入了*实验性*的画板清空功能，为侧边栏加入了动态调整布局的功能，当应用的宽度足够大时，将会自动展开侧边栏并显示描述
